# javascript-app

A javascript application template using pug for a template engine.  Application is transpiled with *babel*, bundled with *webpack*, and styled with with *sass*.

## Build

**javascript-app** is being bundled with webpack. Run `npm run build` to compile production ready files and `npm run watch` during development to automatically compile on file change.

## Usage

Open `dist/index.html` in your browser.

## Test

**javascript-app** uses the Jest testing framework. Run `npm test` to execute all tests.
